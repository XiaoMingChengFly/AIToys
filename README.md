## MVP 能力
- 记一局：局名、日期、备注、玩家与分数录入
- 玩家管理：支持增减玩家，至少保留两人
- 数据校验：玩家名去重、分数不平衡二次确认
- 一键平账：自动调整最后一位已命名玩家分数，使本局总分归零
- 统计页：总局数、参赛人次、累计总分、玩家排行榜
- 历史管理：可删除单局记录或一键清空全部记录

## 快速测试
1. 在“记一局”输入 4 名玩家及分数（总分为 0），点击保存。
2. 切到“统计”，检查总局数与排行榜是否更新。
3. 测试“分数不平衡”保存提示、“一键平账”、单局删除、清空全部记录。
